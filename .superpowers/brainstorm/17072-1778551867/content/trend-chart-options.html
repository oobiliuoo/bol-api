方案 A: 简单柱状图

纯 CSS 柱状图，沿用现有风格，日报/周报/月报切换

特点
  • 无需引入外部库
  • 与现有模型柱状图风格一致
  • 实现简单，代码量小

方案 B: Canvas 折线图

用 Canvas 绘制平滑折线图，显示趋势更直观

特点
  • 趋势展示更清晰
  • 可叠加多个数据系列
  • 实现稍复杂，但效果更好

方案 C: 从零写画图

用原生 JS + Canvas 从零写图表，不依赖外部库

特点
  • 零外部依赖
  • 完全控制样式
  • 实现较复杂
